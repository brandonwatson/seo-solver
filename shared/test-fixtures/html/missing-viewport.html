Missing Viewport Test Page

This page is intentionally missing the viewport meta tag.

SEO validators should detect this and report a no_viewport issue.

Expected Issues

  • no_viewport - Viewport meta tag is missing from the page head

Why This Matters

Without the viewport meta tag, mobile browsers will render the page at desktop width and then scale it down, resulting in a poor mobile experience. Google considers mobile-friendliness a ranking factor.

The Fix

Add the following to the <head> section:

<meta name="viewport" content="width=device-width, initial-scale=1">
